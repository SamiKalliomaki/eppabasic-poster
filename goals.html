Perfect for learning

  • EppaBasic is a web browser based programming language designed for learning programming
  • Language can be used from elementary school to high school
  • Starting programming is as easy as navigating to a web page
  • Nothing is installed on the computer – cloud based technologies ensure code is available anywhere you have an internet connection
  • Instant graphical feedback ensures that the learning process stays motivating
  • Syntax resembling other programming languages provides good base for later transitioning to other languages
